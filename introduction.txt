Did you ever sit in front of your code and failed to understand what it does? Did you ever feel it was too tangled, like spaghetti? And did you ever feel it was hard to add or change features?
These problems are known as Code Smells. Luckily there are already solutions to it: for each code smell there's a set of refactoring steps that fixes it.
This website will show you examples of code smells, and the refactoring steps to fix them. Hopefully this will help you to improve your code, so that in the end you will be delighted by its elegance again.
